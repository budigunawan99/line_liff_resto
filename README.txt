Keterangan:
- Aplikasi ini dapat menambah jumlah makanan/minuman yang ingin dipesan
- Aplikasi ini secara otomatis sudah menghitung jumlah total pesanan
- Aplikasi ini dapat mengirimkan pesan teks & sticker pada ruang chat di mana link dibuka kecuali pada eksternal browser
- Aplikasi ini dapat membagikan pesan teks pada orang-orang yang dipilih
- Aplikasi ini dapat melakukan autentifikasi menggunakan akun LINE, menampilkan nama pengguna dan gambar dari akun LINE
- Aplikasi ini linked dengan LINE OA dari channel LINE Messaging API agar memberikan pilihan untuk menambah OA sebagai teman ketika mengakses link
